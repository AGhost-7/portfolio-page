Projects

Leaps of Sounds

Using a canvas2d “component” that I wrote, I created a website for learning musical scales. Users can create their own instruments, tunings and scales. The website uses RESTful practices (http verbs, error codes, etc) to handle AJAX callbacks. I also added a simple token system for user logins. Database: Postgresql. Other technologies used: Grunt, jQuery UI, AngularJS.
Website Sources

Node Shop

Node Shop is an AngularJS/NodeJS application. The application is a mock online musical instruments store. Other technologies used: Gulp, lodash, Postgresql, ExpressJS and more.
Website Sources

NinjaChat

With Websockets, I implemented a simple chat application which uses the actor model to handle concurrency. The client-side uses jQuery and Bootstrap, with CoffeeScript being the language of choice this time.
